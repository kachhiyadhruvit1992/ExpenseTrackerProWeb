# Expense Tracker Pro Web

Official landing page and legal documentation for **Expense Tracker Pro**, the most intuitive and privacy-focused way to manage personal finances on iOS.

## 🚀 Overview

This repository hosts the static web pages for Expense Tracker Pro. It includes the main landing page, privacy policy, and terms of use, all designed with a clean, modern aesthetic inspired by the iOS design language.

## ✨ Key Features

- **Privacy First**: The app is built with a local-first philosophy. Your financial data stays on your device.
- **Intuitive Tracking**: Log transactions in seconds with smart categorization.
- **Visual Insights**: Beautiful, interactive charts to help you understand spending habits.
- **Data Portability**: Export your records to CSV or Excel anytime for deeper analysis.
- **Local & Secure**: Powered by SwiftData for robust, encrypted local storage.

## 📂 Repository Structure

- `index.html`: The main landing page providing an overview of the app and quick links to legal docs.
- `privacy.html`: Official Privacy Policy detailing our commitment to data security and local storage.
- `terms.html`: Terms of Use governing the app's usage and user responsibilities.
- `legal-styles.css`: The central design system and styling for all documentation pages.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Custom design system with a focus on typography, spacing, and responsive layouts.
- **No Dependencies**: Lightweight and fast-loading with zero external libraries.
